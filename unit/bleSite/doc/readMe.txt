绑定流程：
    1. 获取扫描结果：发布扫描结果消息
    2. 绑定设备：
        1. 发布扫描结果消息
        2. 绑定设备后，更新地址列表、设备列表、状态列表
        3. 发布单个设备绑定成功
        4. 全部成功绑定成功后
                * 同步请求，更新配置站点白名单
                * 发布全部绑定成功

解绑流程；
    1. 删除地址列表、设备列表、状态列表相应条目
    2. 同步请求，删除配置站点白名单中相应条目
    3. 发布设备解绑消息

控制流程：
    1. 接收控制指令
    2. 解析返回状态消息，
    3. 更新状态列表、发布状态变更消息
